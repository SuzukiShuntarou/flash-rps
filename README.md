# flash-rps

コンソール上でフラッシュじゃんけんをするnpmです。

## インストール方法

`$ npm install flash-rps`

## 使用方法

### 開始

`$ npx flash-rps`

### ルール

- フラッシュ形式でCPUがじゃんけんの手を出してきます。
  勝ち、負け、あいこいずれかの手を全て正しく選択出来れば成功です。
  ルールは毎回ランダムに変更されます。

  1.難易度を選択します。Easy,Normal,Hardの3つです。

  2.CPUの選択した手が表示されます。

  3.今回のルールを満たす手を選択します。

  4.全て正しく選択できれば『成功！』と表示されます。
